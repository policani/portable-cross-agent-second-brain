# _inbox — drop raw material here

Drop anything you want the knowledge base to learn from but haven't shaped yet:
pasted notes, exports, transcripts, clippings. This is staging, not memory —
nothing here is trusted.

Run the **ingest** skill to turn items here into proposed notes in `_proposed/`.
After an item is ingested, delete it (or keep it elsewhere as a source of truth).

Don't drop genuinely sensitive material (credentials, personal/legal/medical)
here casually — label it or keep it out.
