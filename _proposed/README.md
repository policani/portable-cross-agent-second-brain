# _proposed — drafts awaiting your approval

The ingest and curate skills write here. Every file is a *proposal*, never
trusted memory. Review each, edit if needed, then promote it into
`knowledge-base/` — or delete it.

Each proposed note carries frontmatter so nothing is mistaken for a fact:

```
---
title: <short title>
source: <file, email, meeting, chat>
date: 2026-06-26
status: extracted | inferred | verified | deprecated
sensitivity: public | internal | confidential | restricted
---
```

The invariant: AI only ever proposes. A human approves before anything becomes
trusted memory.
