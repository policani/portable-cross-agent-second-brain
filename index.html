Portable Cross-Agent Second Brain

A plain-markdown second brain that works the same in Claude, Codex, and Cursor — and routes to the note it needs, so it saves tokens.

One vault, three agents, two files. Claude reads CLAUDE.md; Codex and Cursor read AGENTS.md (Cursor natively). Same rules, phrased per client — switch tools with zero migration.

Saves tokens:  the agent reads a routing map and opens only the note it needs — retrieval without a vector database.

How new knowledge enters

_inbox/ → ingest → _proposed/ → you approve → knowledge-base/

AI only ever proposes. A human approves before anything becomes trusted memory.

Output you can trust

The assistant verifies before it asserts, never hands back placeholder or filler as finished work, and when you push back it re-checks calmly and shows its reasoning instead of doubling down.
